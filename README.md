## ETL With Node Js

### Usage
```sh
    git clone https://github.com/cilas/etl-with-nodejs  
    cd etl-with-nodejs
    npm install
    node index.js
```

### Usage with docker-compose
```sh
    git clone https://github.com/cilas/etl-with-nodejs  
    cd etl-with-nodejs
    docker-compose up -d 
    docker-compose exec app sh
    npm install
    node index.js
```
